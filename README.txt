A pasta notebooks descontinuados contém notebooks que não são versões finais uma vez que foram experiências que falharam, porém são mencionados nos relatórios dos outros notebooks.
A pasta clean contem todos os notebooks sem markdowns e sem outputs.
A pasta pdf contem todos os notebooks transformados em pdf (tanto os com markdowns e ouputs como os sem markdown e sem outputs)

O ficheiro resultados contem uma tabela com os diferentes melhores resultados obtidos nos diferentes modelos ao longo do estudo para uma fácil comparação dos resultados obtidos.

Os modelos que terminam com .h5 em vez de .keras foram treinados no google collab, dai o nome. Estes modelos também descontinuados, dai não ter sido feita alteração no formato deles.
